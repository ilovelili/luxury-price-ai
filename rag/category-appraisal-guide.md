# Category Appraisal Guide

Use this guide for category-specific condition review and missing-input questions. The brand guide handles brand/model cues; this category guide handles what to inspect.

## watches / 時計

Capture:

- Brand, model, reference number, serial if visible
- Case size, material, dial color, bracelet or strap type
- Movement status if known
- Box, warranty card, extra links, service history

Condition risks:

- Case scratches, glass chips, bracelet stretch, crown issues, non-working movement, aftermarket parts

Staff review:

- Always recommend staff or specialist review for high-value watches, unclear reference numbers, missing warranty, or uncertain movement condition.

## bags / ブランドバッグ

Capture:

- Brand, model, shape, size, material, color, hardware, strap/handle, serial or IC marker
- Box, dust bag, card, receipt, pouch, lock/key

Condition risks:

- Corner wear, handle darkening, surface scratches, stains, shape collapse, hardware tarnish, zipper/lock issues, sticky interior, odor if user states it

## precious-metals / 貴金属

Capture:

- Material stamp such as K18, Pt900, Pt850, 925
- Weight, stone details, brand, size, certificate

Condition risks:

- Deformation, missing stones, clasp failure, deep scratches, engraving, certificate mismatch

Staff review:

- Always route to specialist review for material verification, weight, stones, and market metal price.

## accessories / アクセサリー

Capture:

- Item type: necklace, bracelet, ring, earrings, brooch, scarf, belt, sunglasses, key holder
- Size, material, stamp, logo, stone/costume jewelry details, accessories

Condition risks:

- Tarnish, plating wear, missing stones, chain/clasp damage, fabric pulls, lens scratches, belt hole wear

## small-goods / 財布・小物

Capture:

- Model, material, color, hardware, card slots, coin pocket, date code or IC marker

Condition risks:

- Corner wear, edge coating cracks, coin-pocket dirt, zipper pull wear, snap looseness, interior peeling or stickiness

## ceramics / 陶器・食器

Capture:

- Brand, series, item count, dimensions, box, certificate

Condition risks:

- Chips, cracks, crazing, discoloration, missing pieces, repaired areas

Staff review:

- If brand is a luxury fashion brand collaboration, still judge as ceramics condition first.

## cameras / カメラ

Capture:

- Brand, model, lens, serial, shutter count if known, accessories

Condition risks:

- Fungus, haze, dust, sensor damage, lens scratches, battery corrosion, non-working functions

Staff review:

- Recommend specialist confirmation for operation and optics.

## apparel / 服

Capture:

- Item type, size, material tag, season, model name, color, accessories

Condition risks:

- Stains, discoloration, odor if user states it, fraying, holes, missing buttons, zipper issues, tag condition, alterations

## shoes / スニーカー・靴

Capture:

- Model, size, colorway, material, box, receipt, spare laces

Condition risks:

- Sole wear, heel drag, creasing, stains, odor if user states it, hydrolysis, separation, missing accessories

## art / 美術品・骨董品

Capture:

- Artist/maker, title, dimensions, medium, signature, provenance, certificate, box

Condition risks:

- Cracks, fading, stains, frame damage, repairs, missing certificate

Staff review:

- Always recommend specialist review. Do not infer authenticity or price from images alone.

## furniture / 家具

Capture:

- Brand/designer, model, dimensions, material, age, proof of purchase

Condition risks:

- Scratches, stains, sagging, wobble, repairs, missing parts, upholstery damage

## bicycles / 自転車

Capture:

- Brand, model, frame size, components, year, serial, accessories

Condition risks:

- Frame damage, rust, wheel/bearing issues, drivetrain wear, battery condition for e-bikes

## home-appliances / 家電

Capture:

- Brand, model number, year, operation status, accessories, warranty

Condition risks:

- Non-working functions, battery degradation, missing parts, scratches, hygiene concerns

## hobbies / ホビー

Capture:

- Item type, brand, model, scale, completeness, box, limited edition details

Condition risks:

- Missing parts, box damage, discoloration, repair, non-working electronics

## sports / スポーツ用品

Capture:

- Sport type, brand, model, size/spec, accessories

Condition risks:

- Wear, cracks, grip deterioration, rust, safety concerns

## instruments / 楽器

Capture:

- Brand, model, serial, year, accessories, case, operation status

Condition risks:

- Cracks, neck warp, electronics issues, missing parts, repair history

Staff review:

- Recommend specialist review for playability and originality.

## consumables / 消耗品

Capture:

- Brand, product type, unopened/opened, expiration date, storage condition

Condition risks:

- Opened packaging, expired goods, hygiene concerns, liquid leakage

Staff review:

- If resale policy is unclear, route to staff.

## games / ゲーム

Capture:

- Console/software, model, region, accessories, box/manual, operation status

Condition risks:

- Scratches, battery leakage, missing cables, stick drift, non-working drive

## computers / パソコン・PC

Capture:

- Brand, model, CPU, memory, storage, year, serial, battery status, accessories

Condition risks:

- Battery swelling, screen damage, keyboard defects, activation lock, missing charger, storage wipe status

## camping / キャンプ用品

Capture:

- Brand, model, item type, size, accessories, usage count if known

Condition risks:

- Fabric tears, pole damage, stains, mildew risk, missing parts, fuel equipment safety

## audio / オーディオ

Capture:

- Brand, model, serial, accessories, operation status

Condition risks:

- Driver issues, noise, battery degradation, cable damage, missing remote, cosmetic scratches

## Generic Fallback

If the user provides only a vague description:

- Ask for brand, category, model or item name, size, material/color, condition, photos, accessories, purchase timing, and any serial/authenticity marker.
- Call the pricing API only after enough structured fields are available.
- If still insufficient, return a cautious staff-review draft with missing inputs.
