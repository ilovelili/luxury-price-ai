# Brand Model Material Guide

Use this guide to normalize visible product details and write appraisal rationale across the exported luxury brands. This is not a price table. Always use the pricing API for market and purchase-offer ranges.

## LOUIS VUITTON

Common bag and small-goods cues:

- Models: Speedy, Neverfull, Alma, Noe, Keepall, Pochette, OnTheGo, Capucines, Zippy Wallet, Sarah Wallet
- Lines/materials: Monogram, Damier Ebene, Damier Azur, Epi, Taiga, Vernis, Empreinte, Mahina
- Condition risks:ヌメ革焼け, water stains, canvas cracking, corner wear, handle darkening, zipper wear, sticky interior, odor if user states it
- Accessories: box, dust bag, receipt, padlock/key, strap, pouch, date code or IC chip evidence

Useful wording:

- 「ルイヴィトンはライン、サイズ、ヌメ革の焼けやシミ、角スレ、付属品の有無で評価が変わります。」

## HERMES

- Leathers: Togo, Clemence, Epsom, Box Calf, Swift, Taurillon, Ardennes, Barenia

- 「エルメスはモデル、サイズ、革種、刻印、金具色、付属品の有無が評価に大きく影響します。」

## Common Color And Hardware Tokens

Normalize these when visible or user-provided:

- 黒 / ブラック / black
- 白 / ホワイト / white
- ベージュ / beige
- ブラウン / brown
- ネイビー / navy
- 赤 / レッド / red
- ピンク / pink
- ゴールド金具 / gold hardware
- シルバー金具 / silver hardware
- ヴィンテージ金具 / vintage hardware

Mention color only if supported by user input or visible images. If lighting is poor, ask for confirmation.

## Authenticity Caution

Do not state that an item is authentic based only on photos.

Use neutral wording:

- 「真贋を含む最終確認は実物確認後となります。」
- 「シリアル・ICプレート・刻印・縫製・金具などの確認が必要です。」
